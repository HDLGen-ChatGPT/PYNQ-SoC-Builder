# Automated PYNQ Overlay Generation

## Overview 

PYNQ Automate is a CLI tool for generating PYNQ Overlays in Vivado automatically.

## Usage:

```
python pynq_automate.py [command] --hdlgen [path to .hdlgen file]
```

## Availabile Commands

**generate_tcl**
**run_vivado**
